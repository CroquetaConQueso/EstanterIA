Acerca de Suite Tienda

Suite Tienda es la interfaz operativa de EstanterIA: gestión de estanterías, planos, inspecciones, alertas, tareas e inventario en una misma demo funcional.

Objetivo del proyecto

El objetivo principal es estructurar la operación de una estantería a partir de su configuración real y del resultado visual por slots:

  • Detectar slots ocupados, vacíos y anómalos.
  • Cruzar inspecciones con productos esperados y asignaciones activas.
  • Generar alertas y tareas para responsables operativos.

Las vistas privadas requieren iniciar sesión con una cuenta autorizada.

Flujo operativo

La demo cubre el ciclo típico de trabajo de un encargado:

  • Visualizar el estado de la estantería en el plano.
  • Revisar inspecciones IA con imagen, slots y confianza.
  • Atender alertas y tareas derivadas de incidencias.

El flujo permite pasar del plano y la inspección al inventario, la alerta y la tarea operativa sin perder el contexto.

Organización del sistema

El sistema separa la gestión operativa de las evidencias visuales:

  • Los planos muestran secciones, estanterías y slots.
  • Vision aporta resultados visuales para cada inspección.
  • Inventario, alertas y tareas mantienen la trazabilidad operativa.

El foco visual es operativo: lectura rápida, contraste fuerte y navegación directa.

Alcance actual

La aplicación está orientada a demo de PFC, con un dominio acotado:

  • No pretende cubrir un ERP completo.
  • No incluye funciones de nóminas, pedidos ni stock numérico.
  • Prioriza el flujo de inspección, alerta y tarea operativa.

El objetivo es demostrar un flujo coherente de visión, inventario y operación.
