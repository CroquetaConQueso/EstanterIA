Acerca de Suite Tienda

Suite Tienda es un prototipo de aplicación web que combina planos 2D, visión por computador e inventario para ayudar a empresas a controlar sus estanterías y detectar huecos de producto de forma automática.

Objetivo del proyecto

El objetivo principal es estructurar la lógica de un negocio a partir de sus planos y estanterías, y que la IA se encargue de:

  • Detectar qué productos están colocados en cada estantería.
  • Identificar huecos y desajustes entre stock real y stock de sistema.
  • Asignar tareas a los responsables de cada zona o sección.

La interfaz que ves es un boceto funcional pensado para enseñar el flujo de trabajo completo antes de implementar el backend en Spring Boot.

Flujo funcional y alcance

El prototipo cubre el ciclo típico de trabajo de un encargado o reponedor:

  • Visualizar el plano y las estanterías de la tienda.
  • Revisar inspecciones IA con fotos, secciones y huecos detectados.
  • Actualizar el inventario según las acciones realizadas.

Cada pantalla está pensada para conectarse más adelante con endpoints REST: carga de planos, resultados de inspección, estado del stock y tareas asignadas.

Diseño técnico

El diseño del prototipo sigue una estructura similar a la que se usaría en producción:

  • División por vistas: acceso, panel principal, planos, inspecciones, inventario.
  • Uso de JSON de planos como fuente de verdad para el render 2D.
  • Separación entre layout visual y datos, para poder conectar un backend.

El diseño actual ya separa las secciones principales, de forma que sea sencillo conectar cada vista con su correspondiente endpoint del backend más adelante.

Qué NO es este prototipo

Este boceto no es todavía una aplicación completa en producción:

  • No hay autenticación real ni gestión de usuarios.
  • No se persisten datos en base de datos; los contenidos son de ejemplo.
  • La IA no está integrada: las inspecciones se simulan con datos mock.

El objetivo es tener un esqueleto claro para luego conectar la lógica de negocio, la IA y la base de datos sin tener que rediseñar toda la interfaz.
