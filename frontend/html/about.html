Acerca del proyecto

Acerca de EstanterIA

EstanterIA es un sistema web para la gestión operativa de estanterías: planos 2D, inspecciones, inventario, alertas, tareas, equipo e informes en una misma demo funcional.

Objetivo del proyecto

El objetivo principal es estructurar la operación de estanterías a partir de planos 2D, secciones, slots, productos esperados y resultados visuales:

  • Asociar secciones, estanterías, slots y productos.
  • Registrar inspecciones visuales y manuales con trazabilidad.
  • Generar alertas y tareas para responsables operativos.

Las vistas privadas requieren iniciar sesión con una cuenta autorizada.

Flujo operativo

La demo cubre el ciclo principal de trabajo de un administrador o gerente:

  • Configurar planos, zonas y estanterías colocadas.
  • Lanzar inspecciones con apoyo de Vision/FastAPI.
  • Atender alertas, tareas y revisiones de inventario operativo.

El flujo permite pasar del plano y la inspección al inventario, la alerta, la tarea y el informe sin perder el contexto operativo.

Organización del sistema

El sistema separa la gestión operativa de las evidencias visuales:

  • Los planos muestran secciones, estanterías, slots y estados.
  • Vision aporta resultados visuales por slot para cada inspección.
  • Inventario, alertas, tareas y trabajadores mantienen la trazabilidad.

Los informes resumen rotación visual y vaciados detectados; no calculan ventas reales, ingresos ni rentabilidad.

Alcance actual

La aplicación está orientada a demo de PFC, con un dominio acotado:

  • No pretende cubrir un ERP completo.
  • No incluye funciones de nóminas, pedidos ni stock numérico.
  • Conserva histórico mediante baja lógica en entidades operativas.

El objetivo es demostrar un flujo coherente de planos, visión, inventario, equipo, alertas, tareas e informes.
