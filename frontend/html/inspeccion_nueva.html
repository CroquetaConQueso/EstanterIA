Flujo de trabajo

Nueva inspección

Crea una inspección base (sin IA). La imagen y el análisis se enganchan después sin romper el flujo.

Datos de la inspección

MVP: plano + código de estantería + notas. El estado inicial se marca como CREADA.

Volver

Tras crear, se vuelve al listado y se selecciona la inspección nueva.

Qué se crea exactamente

  • Inspección con fecha, plano y estantería.
  • Estado inicial (CREADA) para el flujo.
  • Notas para contexto humano.

Encaje con IA

Cuando metas YOLO, esta inspección será el “contenedor”:

  1. Subir imagen asociada
  2. Lanzar análisis
  3. Guardar resultado
  4. Actualizar estado
