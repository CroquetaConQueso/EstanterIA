Estanterías, visión y alertas en un solo flujo.

Suite Tienda combina plano operativo, inspecciones por slots, inventario real, alertas y tareas para que cada incidencia tenga contexto y responsable.

La demo funcional trabaja con backend Spring Boot, Vision FastAPI y PostgreSQL.

Resumen rápido

  • Plano conectado al estado visual de EST-001.
  • Inspecciones por slots con imagen y confianza.
  • Alertas y tareas operativas trazables.

El dashboard operativo está protegido y requiere iniciar sesión.

Cómo se organiza la operación

  1. 1. Configura la estantería

    Cada slot tiene producto esperado, proveedor y asignación activa cuando procede.

  2. 2. Lanza inspecciones

    Vision clasifica slots como ocupados, vacíos o anómalos y persiste el resultado.

  3. 3. Resuelve incidencias

    Las alertas derivan en tareas operativas para revisión, reposición o retirada.
